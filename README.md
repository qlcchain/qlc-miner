---

Simple QLC miner just for developer testing.

## Build and Run
```shell
make clean build
./gqlc-miner --help
```

## Docker
You can build the docker image yourself or download it from docker hub
### Build docker images

```bash
cd docker
./build.sh
```

### Download docker images from docker hub

```bash
docker pull qlcchain/qlc-miner:latest
```
